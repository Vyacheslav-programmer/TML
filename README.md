## Краткое описание исследования

### Обзор данных
Для анализа использовался датасет `Admission_Predict.csv`, содержащий информацию о 400 кандидатах и 9 признаках, включая академические показатели и вероятность поступления в магистратуру.

### Предобработка данных
- Загрузка данных из файла `Admission_Predict.csv`
- Очистка названий столбцов (удаление лишних пробелов и символов)
- Первичный анализ данных с помощью функций `info()`, `head()`, `describe()`
- Создание пропусков (15% значений в `GRE Score` и `University Rating`)
- Удаление строк с пропусками (осталось 280 записей)

### Анализ данных
- Гистограмма для анализа распределения среднего балла (CGPA)
- Диаграмма рассеяния для анализа связи между баллами GRE и вероятностью поступления
- Корреляционная тепловая карта для выявления взаимосвязей между признаками
- **Boxplot (ящик с усами)** — построен в соответствии с требованием для группы ИУ5Ц-83Б

### Основные выводы
1. Распределение CGPA близко к нормальному (большинство кандидатов имеют баллы 8.5-9.5)
2. CGPA имеет сильную положительную корреляцию с вероятностью поступления (0.87)
3. GRE Score также сильно коррелирует с целевым признаком (0.81)
4. Данные хорошо подходят для решения задачи регрессии

### Использованные библиотеки
- `pandas` — работа с данными
- `numpy` — математические операции
- `matplotlib` — визуализация
- `seaborn` — статистические графики
- `sklearn.preprocessing` — масштабирование данных
